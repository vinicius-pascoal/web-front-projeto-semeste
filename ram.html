o que e ?

A memória RAM (Random Access Memory) é um tipo de memória de computador volátil e temporária usada pelo sistema operacional e aplicativos para armazenar dados e informações temporárias enquanto o computador está ligado. Ela é mais rápida do que o armazenamento permanente, como um disco rígido ou SSD, e permite que o computador acesse rapidamente dados necessários para executar programas e realizar tarefas. Quando o computador é desligado, os dados na RAM são apagados, ao contrário do armazenamento permanente, que retém informações mesmo quando o computador está desligado.

necessita de :

A memória RAM necessita de energia elétrica constante para manter os dados armazenados temporariamente. Quando o computador é desligado, os dados na RAM são perdidos, por isso a necessidade de energia contínua para manter as informações durante o uso do computador.

e incompativel com:

A memória RAM pode ser incompatível com um computador devido a diferenças em especificações como tipo (DDR3, DDR4, etc.), capacidade, frequência (MHz) ou configuração física (por exemplo, DIMM vs. SO-DIMM). Se as especificações da RAM não correspondem aos requisitos da placa-mãe ou do processador, pode ocorrer incompatibilidade. Além disso, sistemas operacionais também podem ter limites de capacidade de RAM.
